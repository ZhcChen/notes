## 🌟 Flutter 特点

- **跨平台**：一次编写，部署到 iOS、Android、Web 和桌面
- **高性能**：使用 Dart 和 Skia 图形引擎，提供接近原生的性能
- **热重载**：快速迭代开发，无需重新启动应用
- **丰富的组件库**：提供大量预构建的 Material Design 和 Cupertino 风格组件
- **强类型语言**：Dart 语言提供静态类型检查和可靠的代码
